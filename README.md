# Artfolio

Artfolio is an Android application designed to curate and display art collections. It creates a seamless experience for users to browse, manage, and view artworks effectively.

## Features

- **Splash Screen**: Welcoming launch experience.
- **Art Gallery**: Browse a collection of artworks.
- **Local Storage**: Persist data using Room Database.
- **Smooth Image Loading**: Optimized image rendering with Glide.
- **Modern Navigation**: Seamless transitions using Jetpack Navigation Component.

## Tech Stack

- **Language**: Kotlin
- **UI Toolkit**: XML Layouts with Material Design 3
- **Architecture**: Modern Android Architecture components
- **Dependencies**:
  - [Jetpack Navigation](https://developer.android.com/guide/navigation) - For in-app navigation
  - [Room Database](https://developer.android.com/training/data-storage/room) - For offline data persistence
  - [Glide](https://github.com/bumptech/glide) - For image loading and caching
  - [ViewBinding](https://developer.android.com/topic/libraries/view-binding) - For safer view interaction

## Setup & Installation

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Open in Android Studio**:
    - Launch Android Studio.
    - Select "Open an existing Android Studio project".
    - Navigate to the cloned directory and select it.
3.  **Sync Gradle**:
    - Allow Android Studio to sync the project and download detailed dependencies.
4.  **Run the App**:
    - Connect an Android device or start an emulator.
    - Click the "Run" button (green play icon).

## Requirements

- Android Studio Iguana or newer recommended.
- Minimum SDK: API 21 (Android 5.0 Lollipop).
- Target SDK: API 34 (Android 14).
